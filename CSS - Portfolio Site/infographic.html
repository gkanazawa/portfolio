Electric Vehicle Longform Infographic


This was my first project in UW's Graphic Design in Cartography, a longform infographic utilizing ArcGIS Pro and Adobe Illustrator. It features a bivariate map combining rates of EV ownership enumerated by state, along with proportional symbols representing the number of "fast-charge" charging stations normalized by population.
